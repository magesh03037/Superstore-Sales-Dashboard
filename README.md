# 📊 Superstore Sales Performance Dashboard

A data analytics project analyzing 4 years of retail sales data using Power BI.

## 🌟 Features
- 📈 Interactive Power BI Dashboard
- 🔍 Sales trend analysis 2014-2017
- 🏷️ Category performance analysis
- 🗺️ Regional sales distribution
- 👥 Customer segment analysis
- 🎯 KPI cards and slicers

## 📁 Files
- **mag sales excel.xlsx** — Cleaned dataset
- **magesh sales and revenue project.pbix** — Power BI Dashboard
- **magesh sales and revenue ppt.pdf** — Presentation

## 📊 Key Insights
- Total Sales: 2.30M
- Total Profit: 286.40K
- Total Quantity: 38K
- Technology is top performing category
- East region leads in sales

## 🛠️ Tools Used
- Microsoft Excel — Data cleaning
- Power BI — Dashboard creation
- Kaggle — Dataset source
